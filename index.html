这是什么

基于 magic-wormhole.rs 的 Windows 桌面 GUI 工具。一方点 发送 得到一个短码（如 15-123-456）， 通过电话或当面告诉另一方；另一方点 接收 输入短码就建立加密通道。

没有账号、没有手机号、没有联系人，关闭即销毁。 握手后双方可以发任意数量的文字和文件。

🔒

端到端加密

PAKE + NaCl SecretBox + Noise transit，全程在你和对方两台设备之间。

📁

任意类型文件

拖拽或选择，进行中可取消。可执行类文件红色二次确认。

🌐

自定义 relay

默认 Least Authority 公共 relay；设置里可改成你自建的服务器。
